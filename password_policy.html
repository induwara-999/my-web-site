Strong Password Policy

To ensure the security of your account, all users must create passwords in compliance with the following requirements.

Password Requirements

  • Length: Must be at least **12** characters long.
  • Complexity: Must contain at least **three** of the following four character types:
    • Uppercase letters (A, B, C...)
    • Lowercase letters (a, b, c...)
    • Numbers (0, 1, 2...)
    • Special characters (!, @, #, $, %...)
  • Common Passwords Prohibited: Avoid using common passwords such as "password," "123456," or your name.
  • Time Limit: It is recommended to change the password every **90** days for security.
  • Reuse: The last 5 used passwords cannot be reused.

Security Tips

It is highly recommended to use a Password Manager application to manage a password.
